# IBM HR Employee Attrition Analysis

## Project Overview
This project analyzes employee attrition patterns using SQL.
The goal was to identify key factors influencing employee turnover.

## Tools Used
- MySQL

## SQL Concepts Applied
- Data Cleaning & Validation
- Aggregations (COUNT, AVG)
- CASE-based segmentation
- Window Functions (RANK, NTILE)
- Attrition rate calculations

## Key Insights
- Higher attrition among employees working overtime
- Lower salary groups show significantly higher turnover
- Attrition decreases with seniority and experience
- Sales department shows highest attrition risk
- Freshers are more likely to leave compared to long-tenured employees
